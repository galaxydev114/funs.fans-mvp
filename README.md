# funs.fans homologation
Porn content creating & sharing
 
## Get Started
env setting

Composer install
```sh
$ composer install
```

Laravel license
```sh
$ php artisan key:generate
```

Database migrate and seed
```sh
$ php artisan migrate --seed
```

Storage link
```sh
$ php artisan storage:link
```

Run serve
```sh
$ php artisan serve
```
